# AI and Learning English Quiz

## Overview
This repository contains an **interactive quiz** designed to help students understand what AI can and cannot do when it comes to learning English. The quiz is based on real-world scenarios and limitations of AI in language learning.

## Features
- Mobile-friendly responsive design
- Shuffled questions and options each time the page loads
- Progress bar to track completion
- Instant feedback for each question
- Motivational messages based on score
- Displays correct answers after submission

## How to View the Quiz
1. Upload the file `AI_Extended_Quiz_Engaging.html` to this repository.
2. Enable **GitHub Pages**:
   - Go to **Settings** > **Pages**.
   - Under **Source**, select `main` branch and `/root` folder.
   - Click **Save**.
3. GitHub will provide a public URL like:
   ```
   https://yourusername.github.io/AI-English-Quiz/AI_Extended_Quiz_Engaging.html
   ```
4. Share this link with students. They can open it on any device (phone, tablet, PC).
